# Δ-controller

Δ-controller is the Kubernetes application layer in this repository. It runs as
a JavaScript program inside the `dbsp` runtime and executes declarative
incremental controller pipelines.

Documentation lives in `/doc/`:

- [Sources, targets, and pipelines](/doc/apps-dctl-sources-targets-pipeline.md)
